可上传的最大文件大小取决于 Organization.MaxUploadFileSize 属性。 此属性位于 Dynamics 365 应用程序中“系统设置”的“电子邮件”选项卡中。 此设置会限制可附加到电子邮件、注释和 Web 资源的文件大小。 默认设置为 5 MB。
