若要确保用户在单击“帮助”按钮时获得正确的信息，需要将 [!INCLUDE[pn_crm_shortest](pn-crm-shortest.md)] 设置为使用自定义帮助。 转到“设置” > “管理”，依次单击“系统设置”和“常规”选项卡，然后在“为可自定义实体使用自定义帮助”中选择“确定”。
