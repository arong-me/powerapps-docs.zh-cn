# <a name="additional-controls-for-dynamics-365-for-phones-and-tablets"></a>适用于手机和平板电脑的 Dynamics 365 的附加控件 

 可使用一组丰富的附加控件在适用于手机和平板电脑的 Dynamics 365 上创造更友好的触摸体验。 这些控件包括滑块、开关、多媒体播放器、输入掩码、日历及其他控件。  

 
> [!NOTE]
>  只能将这些附加控件用于移动应用。 Web 应用不支持这些控件。  
  
 若要在窗体编辑器中使用这些控件：  
  
1.  双击要添加控件的字段或列表。  
  
2.  单击“控件”选项卡。  
  
3.  单击“添加控件”。  
  
4.  选择所需控件，然后单击“添加”。  
  
    > [!NOTE]
    >  根据字段或列表类型，可使用不同的控件。 例如，滑块控件可能仅适用于数字或货币字段，而日历控件仅适用于列表。  
  
5.  选择要在其上显示控件的设备（手机、平板电脑或这两者）。 控件不适用于手机标头字段。  
  
6.  配置每个属性的值。  
  
7.  完成控件配置后，单击“确定”。  
  
 以下是你可在手机和平板电脑的 Dynamics 365 窗体上使用的每个控件的说明。  
  
## <a name="calendar-control"></a>日历控件  
 使用此控件配置窗体，使其在手机和平板电脑的 Dynamics 365 中显示为日历视图。 还可使用此控件替换手机和平板电脑的仪表板、列表或实体网格。  
  
|属性|描述|  
|--------------|-----------------|  
|开始日期|定义要在日历视图中可视化的项的开始日期和时间。 可用值为此视图中日期类型的任何列。|  
|结束日期|定义要在日历视图中可视化的项的结束日期和时间。 可用值为此视图中日期类型的任何列。|  
|持续时间|持续时间（分钟）。 如果为“结束日期”指定了值，则会忽略“持续时间”。|  
|描述|这是要查看的日历项的字幕。|  
  
 日历中显示的最短持续时间为 30 分钟。 持续时间少于 30 分钟的项仍将显示为 30 分钟。  
  
 日历控件支持所有日期行为（“用户当地时间”、“仅日期”和“时区无关”）。

|CC_Timeline_Label1_Desc|标签 1 的说明。|  
|CC_Timeline_Label2|要在标签 1 后显示的字段。|  
|CC_Timeline_Label2_Desc|标签 2 的说明。|  
|CC_Timeline_Label3|要在标签 2 后显示的字段。|  
|CC_Timeline_Label3_Desc|标签 3 的说明。|  
|CC_Timeline_Label4|要在标签 3 后显示的字段。|  
|CC_Timeline_Label4_Desc|标签 4 的说明。|  
|CC_Timeline_Label5|要在标签 4 后显示的字段。|  
|CC_Timeline_Label5_Desc|标签 5 的说明。|  
|CC_Timeline_Timestamp|用于按倒序时间顺序对时间线进行排序的字段。|  
|CC_Timeline_Timestamp_Desc|时间戳的说明。|  
|CC_Timeline_Group|用于对时间线进行分组的映射字段。|  
|CC_Timeline_Group_Desc|“组”字段的说明。|  
|CC_Timeline_GroupOrder|项所属的组相对于其他组的顺序（为要显示的组分配值 1、2、3 等）。 该组将以分配的组值的升序显示。|  
|CC_Timeline_GroupOrder_Desc|“组顺序”字段的说明。|  
|CC_Timeline_URL|要映射的 URL 字段，用于显示每个时间线项的 URL。|  
|CC_Timeline_URL_Desc|URL 字段的说明。|  
|CC_Timeline_ThumbnailURL|用于要为每个项显示的映像/图标缩略图的映射字段。|  
|CC_Timeline_ThumnailURL_Desc|`ThumbnailURL` 字段的说明。|  
|CC_Timeline_Filter|用于时间线筛选器的映射字段。|  
|CC_Timeline_Filter_Desc|筛选器的说明。|  
|CC_Timeline_Footer|要显示为时间线页脚的 Web 资源。|  
|CC_Timeline_Footer_Desc|“页脚”字段的说明。|  
  
## <a name="linear-slider"></a>线性滑块  
 线性滑块控件允许用户通过拖动滑块输入数值，并提供输入数量的选项。 滑块仅提供整数输入和显示。 此控件可用于任何数字或货币字段。  
  
|属性|描述|  
|--------------|-----------------|  
|最多|设置要在滑块上显示的最大值。|  
|Min|设置要在滑块上显示的最小值。|  
|值|要在滑块上显示的值。|  
|步骤|使用此控件输入数据时，设置要从当前值中增减的量。|

 选项集控件提供了一组选项供用户在输入数据时进行选择。 此控件可用于仅包含两个或三个选项的选项集。  
  
|属性|描述|  
|--------------|-----------------|  
|字段|显示控件映射到的字段。|

 翻转开关类似于打开/关闭开关，提供两个值之间的选择。  
  
|属性|描述|

## <a name="star-rating"></a>星级评分  
 使用星级评分可提供评级的视觉表示形式。 可设置的最大星数为 5。 只能对整数使用此控件；它不接受十进制值。  
  
> [!NOTE]
>  请确保为此控件选择“在 Web 上隐藏”选项。  
  
|属性|描述|  
|--------------|-----------------|  
|最多|从下拉列表中选择控件的最大星数。|  
  
## <a name="radial-knob"></a>星形旋钮  
 星形旋钮为用户提供了一种通过滑动旋钮输入数据的方式，并在屏幕上显示为圆形。 星形旋钮控件仅提供整数输入和显示。 此控件可用于任何数字或货币字段。 可使用触摸更改值，也可使用键盘聚焦数字并对其进行编辑。  
  
> [!NOTE]
>  Android 4.2 和 4.3 设备不支持此控件。 它会影响这些版本的滚动体验。  
  
|属性|描述|  
|--------------|-----------------|  
|最多|设置要在仪表上显示的最大值。|  
|Min|设置要在仪表上显示的最小值。|  
|值|获取或设置要在仪表上显示的值。|  
|步骤|使用此控件输入数据时，设置要从当前值中增减的量。|

## <a name="bullet-graph"></a>子弹图  
 子弹图控件显示一个带有比较度量和定性范围的单个关键度量，以即时指示该度量是良好、糟糕还是处于另一状态。 可在仪表板中将此控件用于任何数字或货币字段。 例如，可将值映射到实际收入，将目标映射到估算收入，以直观显示实际收入和估算收入。  
  
|属性|描述|  
|--------------|-----------------|  
|最多|设置要在图上显示的最大值。|  
|Min|设置要在图上显示的最小值。|  
|良好|设置一个被认为适合该度量的值（可选）。|  
|糟糕|设置一个被认为不适合该度量的值（可选）。|  
|值|显示控件映射到的字段。|  
|目标|将此属性映射到要与之比较值的字段。 例如，如果“值”映射到“实际收入”，可将“目标”映射到“估算收入”，或者可提供静态值。|  
  
## <a name="pen-control"></a>笔控件  
 使用笔控件可捕获已写入的输入内容，例如签名。  
  
> [!NOTE]
>  为此控件映射到的字段所指定的建议的最小“最大长度”为 15000。  
>   
>  请确保为此控件选择“在 Web 上隐藏”选项。  
  
|属性|描述|  
|--------------|-----------------|  
|PenMode|指定“PenMode!Draw”、“PenMode!Erase”或“PenMode!Select”，以确定当用户拖动笔控件中的指针设备时会发生什么情况。|  
  
## <a name="auto-complete"></a>自动完成  
 自动完成控件在你键入时会筛选项列表，并允许你从下拉列表中选择一个值。 例如，可使用此控件让用户从州或国家/地区的下拉列表中进行选择。 此控件映射到“单行文本”类型字段。  
  
|属性|描述|  
|--------------|-----------------|  
|字段|显示控件映射到的字段。|  
|源|设置数据的源（“分组的选项”、“选项集”或“视图”）。|  
|选项集|选择此字段的选项集。|  
|查看|选择此字段的实体和视图。|  
|字段|选择要用作数据源的视图主实体的字段。|  
  
## <a name="multimedia"></a>多媒体  
 可嵌入视频，为外出的销售人员和现场人员提供更丰富的客户体验。 使用此控件可映射到包含要在控件中播放的音频或视频链接的 URL 字段。  
  
> [!NOTE]
>  Android 版本 4.4 及更高版本均支持此控件。  
>   
>  Windows 8 和 Windows 8.1 平板电脑和手机目前不支持 YouTube 视频。 Windows 10 仅支持 HTTPS 视频（包括 YouTube）。  
  
 受支持的媒体类型包括：  
  
-   流式处理 MP4 文件  
  
-   YouTube 视频  
  
-   Azure 媒体  
  
-   音频流  
  
 [隐私声明](use-the-form-editor-legacy.md#BKMK_PrivacyNotices)  
  
|属性|描述|  
|--------------|-----------------|  
|媒体|输入要在此控件中播放的媒体的 URL。|  
  
## <a name="number-input"></a>数字输入  
 使用数字输入控件可帮助用户快速输入数据。 用户只需点击加号和减号按钮，即可按所设置的增量更改数值。 此控件可用于任何数字或货币字段。 用户还可直接在字段中键入数字。 此字段仅在编辑模式下受支持。  
  
|属性|描述|  
|--------------|-----------------|  
|步骤|使用此控件输入数据时，设置要从当前值中增减的量。|  
|字段|显示控件映射到的字段。|  
  
## <a name="input-mask"></a>输入掩码  
 使用输入掩码控件，可设置字段的格式（如电话号码或信用卡等），以防止输入无效数据。 例如，如果你希望用户以 +1-222-555-1011 格式输入美国电话号码，请使用输入掩码 +1-000-000-0000。  
  
|属性|描述|  
|--------------|-----------------|  
|掩码|输入用于在用户输入数据时验证数据的掩码。 可将以下字符组合用于掩码：<br /><br /> 0 - 数字<br /><br /> 9 - 数字或空格<br /><br /> # - 数字、符号或空格<br /><br /> L - 字母<br /><br /> I - 字母或空格<br /><br /> A - 字母数字<br /><br /> A - 字母数字或空格<br /><br /> < - 将后面的字符转换为小写字母<br /><br /> > - 将后面的字符转换为大写字母<br /><br /> &#124; - 禁用大小写转换<br /><br /> \ - 转义任何字符，将其转换为文本<br /><br /> 所有其他 - 文本|  
|字段|显示控件映射到的字段。|
