# <a name="unified-interface-playbook"></a>统一接口 Playbook

本 Playbook 旨在帮助客户、合作伙伴和 Microsoft 现场角色计划和执行从旧 Web 客户端到统一接口的转换。

终止，统一接口 Playbook 文档将帮助您：
- 了解统一接口。 
- 了解转换过程。
- 开发讨论和转换的策略。
- 确保顺利转换。
- 高效管理用户查询和潜在异议。
- 查找将支持此过程的资源。
