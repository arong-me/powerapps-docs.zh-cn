# <a name="entity-overview"></a>实体概述

实体用于建立业务数据模型和管理业务数据。 在开发应用程序时，可以使用标准实体、自定义实体或混合使用。 Common Data Service 默认情况下提供标准实体。 这些内容根据最佳实践进行了设计，以获取组织内最常见的概念和情形。
