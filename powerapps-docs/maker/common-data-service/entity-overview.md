# <a name="entity-overview"></a>安全性概览

实体用于建模和管理业务数据。 开发应用时，可以使用标准实体、自定义实体或将两者一起使用。 CDS for Apps 在默认情况下提供标准实体。 这些实体根据最佳做法设计，可在组织中获取最基本的概念和方案。
