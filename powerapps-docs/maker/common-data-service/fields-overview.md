# <a name="fields-overview"></a>字段概述

字段定义可用于在实体中存储信息的各个数据项。 现有标准实体没有满足要求的字段时，创建新字段来捕获数据。 创建新字段后，请务必将其包含在实体的相应窗体和视图中，以便可用于应用中。
