# <a name="sign-up-for-a-free-license-to-try-powerapps"></a>注册试用 PowerApps 的免费许可证
若要在 PowerApps 中创建应用，需要提供不同级别功能的多个许可证类型之一。 如果您的组织已经为您购买了若干 Office 365 或 Dynamics 365 版本中的任何一个，或您的组织为您购买了 PowerApps 计划 1 或计划 2 许可证，那么您已经有了 PowerApps 的许可证。

> [!NOTE]
> 如果您是管理员，请参阅[为您的组织购买 PowerApps](../administrator/signup-for-powerapps-admin.md) 或[您的组织中的 PowerApps 常见问题](../administrator/signup-question-and-answer.md)。

您可以了解[哪些许可证包括 PowerApps](https://powerapps.microsoft.com/pricing/)，以及它们提供什么。 如果依然无法确定您是否已有许可证，打开 [PowerApps 站点](http://powerapps.microsoft.com)，然后选择**登录**。

![PowerApps 的登录选项](./media/signup-for-powerapps/sign-in.png)

如果您没有许可证或您需要比当前许可证提供的更多的功能，您可以注册 PowerApps 计划 2 的免费试用版。 此许可证提供所有 PowerApps 功能，但会在 30 天后过期。

## <a name="upgrade-an-existing-license"></a>升级现有许可证 
您可以通过使用现有的许可证登录到 PowerApps 然后尝试这些功能中的任何功能来开始免费的 30 天 PowerApps 计划 2 的试用：

* [创建和管理 Common Data Service 的实例](../administrator/signup-for-powerapps-admin.md)。
* 使用 Common Data Service [对您的数据建模](./common-data-service/data-platform-intro.md)。
* 从 PowerApps 管理中心[创建和管理环境](../administrator/environments-administration.md)。
* 在 PowerApps 中[管理本地数据网关](./canvas-apps/gateway-management.md)。  

如果在没有 PowerApps 计划 2 许可证的情况下尝试这些功能中的任一功能，则将提示您开始 30 天的免费试用。 只需选择**开始我的试用**，然后在显示的对话框中同意使用条款。

![同意使用条款](./media/signup-for-powerapps/trial-consent.png)

## <a name="get-a-license-from-scratch"></a>从头获取许可证
如果没有 PowerApps 的许可证，您可以从 [PowerApps 站点](http://powerapps.microsoft.com)注册 PowerApps 计划 2 的 30 天免费试用。

* 如果您是在手机上使用浏览器，点按右上角的菜单图标，然后点按**免费注册**。
* 如果您在其他设备类型上使用浏览器，请选择右上角的**免费注册**。

    ![注册免费试用](./media/signup-for-powerapps/sign-up-browser.png)

在屏幕中间附近，选择**定价**，然后选择**开始免费试用**。

![开始免费试用](./media/signup-for-powerapps/start-free-trial.png)

在显示的对话框中，键入或粘贴您的工作或学校电子邮件地址，然后单击或点按**提交**。

> [!IMPORTANT]
> 不支持消费电子邮件服务、电信提供商、政府和军事地址，包括：  
> 
> * outlook.com、hotmail.com、gmail.com，等等  
> * .gov 和 .mil
>
> 如果尝试使用个人电子邮件地址注册，一条消息将指示必须使用工作或学校电子邮件地址。

![键入电子邮件地址](./media/signup-for-powerapps/get-started-old.png)

如果 PowerApps 识别您的组织，您可以使用该组织的凭据登录（跳过其余说明内容）。

![使用工作帐户登录](./media/signup-for-powerapps/we-know-you.png)

否则，将提示您检查您的电子邮件。 

![要求检查电子邮件的消息](./media/signup-for-powerapps/check-email.png)

在电子邮件中，选择验证您的电子邮件地址的链接，并（如果提示）提供更多信息。

在该对话框显示时，选择**开始**，然后开始使用 [PowerApps](http://web.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc)！

![单击开始](./media/signup-for-powerapps/almost-there.png)

> [!NOTE]
> 如果您希望使用更多时间来评估 PowerApps，可以请求延长另外 30 天试用期。

## <a name="faq"></a>常见问题解答

### <a name="what-premium-features-does-the-trial-license-include"></a>试用许可证包括哪些高级功能？

* **访问 Common Data Service**，这内置于 PowerApps 的安全业务数据平台。 [Common Data Service](./common-data-service/data-platform-intro.md) 附带数百个标准业务实体，所以您的应用可以共享相同的“客户”、“产品”，“潜在顾客”等定义。
* **访问高级连接**，如 Salesforce、DB2、Zendesk 和 Common Data Service。 通过许多 Office 365 和 Dynamics 365 计划，您可以连接到数据源，如 Office 365、Dynamics 365、Dropbox 和 Twitter。 使用试用许可证，还可以通过这些高级连接器及很多其他功能连接到数据：

    ![](./media/signup-for-powerapps/premium-connectors.png)

    在[连接器的完整列表](./canvas-apps/connections-list.md)中，高级连接器将显示并带有徽章：

    ![](./media/signup-for-powerapps/premium-badge.png)
* **访问 PowerApps 管理中心**，在这里，您可以管理您的环境、数据库、用户权限和数据策略。 [了解更多](../administrator/admin-guide.md)
* **在模型驱动模式中创建应用**，在这里，您可以向应用添加组件并让 PowerApps 创建其布局及其他界面元素。 

有关试用许可证的功能和产能的详细信息，请参见[定价页](https://powerapps.microsoft.com/pricing/)。

### <a name="what-happens-when-my-trial-expires"></a>我的试用过期会发生什么情况？
在 30 天后，将提示您请求延长试用期或购买计划。 您可以在[定价页](https://powerapps.microsoft.com/pricing/)找到所有计划的详细信息。

* 如果您可以通过 Office 365、Dynamics 365 或 PowerApps 计划 1 许可证访问 PowerApps，您仍可以使用 PowerApps，但是，您可能会失去对 Common Data Service、高级连接器、PowerApps 管理中心以及 PowerApps 计划 2 的其他功能的访问权限。 例如，如果您尝试创建高级连接，此屏幕可能出现：

    ![](./media/signup-for-powerapps/premium-trial-expired.png)

* 如果您仅有的 PowerApps 访问权限是通过试用（您在 [PowerApps 站点](http://powerapps.microsoft.com/)或[定价页](http://powerapps.microsoft.com/pricing)注册的）获得，您可以请求 30 天的试用许可证延期以保留对 PowerApps 的访问。

### <a name="when-will-my-trial-license-expire"></a>我的试用许可证何时到期？
要确定您的试用许可证何时到期，请选择 [PowerApps 站点](http://web.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc)右上角附近的齿轮图标，然后选择**计划**。

![](./media/signup-for-powerapps/show-duration.png)

### <a name="what-happens-to-my-data-if-my-trial-license-expires"></a>如果我的试用许可证到期，我的数据会发生什么情况？
如果您仍有权访问 PowerApps，则可以继续使用它。 Common Data Service 中的所有数据都将保持不变，将 Common Data Service 用作数据源的任何应用或流将继续照常运行。 但是，您将不能使用该应用或流，如果您尝试在 Common Data Service 中修改架构或实体，系统会提示您请求 30 天的试用延期或购买计划。

### <a name="what-should-i-do-next"></a>我下一步该做什么？
您可以通过以下方法之一保留对 PowerApps 及其功能的访问：

* 在出现提示时将试用许可证延长 30 天。
* 在 PowerApps [定价页](https://powerapps.microsoft.com/pricing/)上查看可用计划，然后[购买一个](../administrator/signup-for-powerapps-admin.md)。
