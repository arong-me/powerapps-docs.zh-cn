# <a name="exit-function-in-powerapps"></a>PowerApps 中的 Exit 函数
退出当前运行的应用。

## <a name="description"></a>说明
**Exit** 函数用于退出当前运行的应用。  退出后，用户会返回应用列表，并且可在列表中选择要打开的另一个应用。

## <a name="syntax"></a>语法
**Exit**()
