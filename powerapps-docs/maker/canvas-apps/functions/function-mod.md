# <a name="mod-function-in-powerapps"></a>PowerApps 中的 Mod 函数
返回除法运算的余数。

## <a name="description"></a>描述
**Mod** 函数可用于返回两个数字相除的余数。

结果的符号与除数的符号相同。

## <a name="syntax"></a>语法
**Mod**( *Number*, *Divisor* )

* *Number* - 必需。 被除数。
* *Divisor* - 必需。  除数。
