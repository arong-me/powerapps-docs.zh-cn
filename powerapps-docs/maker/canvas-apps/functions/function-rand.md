# <a name="rand-function-in-powerapps"></a>PowerApps 中的 Rand 函数
返回一个伪随机数。

## <a name="description"></a>描述
**Rand** 函数返回一个大于或等于 0 但小于 1 的伪随机数。

## <a name="syntax"></a>语法
**Rand**()
