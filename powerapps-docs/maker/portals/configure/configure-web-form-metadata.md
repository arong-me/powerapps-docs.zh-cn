# <a name="configure-web-form-metadata-for-portals"></a>配置门户的 Web 窗体元数据

Web 窗体元数据包含其他行为修改逻辑，以增加或替代不可能通过本地实体窗体编辑功能操作的窗体字段的功能。

## <a name="add-a-new-record"></a>添加新记录

1. 在具有您要修改的字段的 Web 窗体步骤中，转到**相关** > **元数据** 

2. 选择**新建 Web 窗体元数据**。

## <a name="web-form-metadata-properties"></a>Web 窗体元数据属性

以下属性提供窗体中元素的其他样式和功能。

| 客户          | 说明                                                                                                                                                                                                                                                                                                                                          |
|---------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Web 窗体步骤 | 与 Web 窗体元数据记录关联的 Web 窗体步骤。                                                                                                                                                                                                                                                                                      |
| 类型​​          | 可用选项是：<ul><li>属性</li><li>节​​</li><li>Tab</li></ul>选择“属性作为类型”值显示修改为相关步骤呈现的当前窗体中的字段的相应选项。 选择“部分作为类型”值显示修改窗体中的部分的可用选项。 选择“选项卡作为类型”值显示修改窗体中的选项卡的可用选项。  |

## <a name="web-form-metadata-type--attribute"></a>Web 窗体元数据类型 = 属性

在选择的类型为“属性”时，以下属性将显示。


|          客户          |                                                                                                                                                    说明                                                                                                                                                     |
|------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 属性逻辑名称 |                                                                                                                              要修改的属性字段的逻辑名称。                                                                                                                               |
|         标签          | 使用在此输入中指定的文本替换分派给实体中的属性的默认标签。 对于为 Common Data Service 环境安装和启用的每种语言包，将提供一个字段用以使用相关语言输入消息。 |

### <a name="control-style"></a>控制样式

以下选项修改属性字段的样式和功能。


|                      客户                       |                                                                                                                                                                                                                                                                                                                                       说明                                                                                                                                                                                                                                                                                                                                       |
|-------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|                      样式                      | 以下版本之一：<ul><li>作为垂直单选按钮列表的选项集</li><li>作为水平单选按钮列表的选项集</li><li>作为地理位置查找验证程序的单行文本（需要 [!INCLUDE[pn-bing](../../../includes/pn-bing.md)] 地图设置 - 详细信息位于此处）</li><li>作为固定总和的组整数（需要组名称）</li><li>作为等级次序比例禁止绑定的组整数（需要组名称）</li><li>作为等级次序比例允许绑定的组整数（需要组名称）</li><li>多选矩阵（需要组名称）</li><li>多选（需要组名称）</li><li>作为堆栈级别的组整数（需要组名称）</li></u> |
|                   组名称                    |                                                                                                                                                                                                                                                                                                             用于将控件分组到一起作为组合控件的名称。                                                                                                                                                                                                                                                                                                              |
| 多选需要的最少选择计数 |                                                                                                                                                                                                                                                                      这是在多选问题中需要选择的最小值。 仅当选择“多选”控件样式时才必需。                                                                                                                                                                                                                                                                       |
|       多选最大选择数量        |                                                                                                                                                                                                                                                          这是在多选问题中允许选择的值的最大数目。 仅当选择“多选”控件样式时才必需。                                                                                                                                                                                                                                                          |
|           固定总和最小总数            |                                                                                                                                                                                                                                                             这是应用于固定总和响应字段的所需最小值。 仅当选择了“作为固定总和的组整数”控件样式时才是必需的。                                                                                                                                                                                                                                                              |
|           固定总和最大总数            |                                                                                                                                                                                                                                                 这是允许应用于固定总和响应字段的值的最大数目。 仅当选择了“作为固定总和的组整数”控件样式时才是必需的。                                                                                                                                                                                                                                                 |
|           随机化选项集值           |                                                                                                                                                                                                                                                                     指定“是”将对为选项集控件列出的选项随机排序。 仅适用于类型为“选项集”的属性。                                                                                                                                                                                                                                                                     |
|                    CSS                    |                                                                                                                                                                                                                                                                                                                      添加自定义 CSS 类名到控件。                                                                                                                                                                                                                                                                                                                       |

### <a name="prepopulate-field"></a>预填充字段

以下选项为窗体中的字段提供默认值。


|         客户         |                                                                                                                                                                                                                                                                说明                                                                                                                                                                                                                                                                 |
|----------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 忽略默认值 |                                                                              忽略指定属性字段的默认值。 对两个呈现为“是”和“否”单选按钮的选项字段的属性非常有用。 因为默认自动分派值“是”或“否”，所以可通过此选项显示不带预定义响应的判断问题。                                                                               |
|         Type         | 以下版本之一：<ul><li>值</li><li>今天的日期</li><li>当前用户的联系人</li></ul>选择值要求在加载窗体时在将分派给字段的**值**字段中指定值。 选择今天的日期将当前日期和时间分派到属性字段。 选择当前用户的联系人需要是联系人实体中的属性的**窗体属性**，该实体将从当前用户的联系人记录中检索并在指定的属性字段中设置。 |
|        值         |                                                                                                                                                                                                                                        在加载窗体时要分派至字段的值。                                                                                                                                                                                                                                        |
|    从属属性    |                                                                                                                                                                                             将从当前门户用户的记录检索并在加载窗体时分派给字段的联系人实体中的属性。                                                                                                                                                                                             |

### <a name="set-value-on-save"></a>设置保存时的值

以下选项指定在保存窗体时将设置的值。

| 客户              | 说明                                                                                                                                                                                                                                                                                                                                                                                                                                |
|-------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 设置保存时的值 | “是”表示值应分派到使用**值**字段中提供的输入的属性。 <br>**注意**：除以下类型外所有属性类型均支持：唯一标识符。       |
| Type              | 以下版本之一：<ul><li>值</li><li>  今天的日期</li><li>当前用户的联系人</li></ul>选择值要求在加载窗体时在将分派给字段的**值**字段中指定值。 选择今天的日期将当前日期和时间分派到属性字段。 选择当前用户的联系人需要是联系人实体中的属性的**窗体属性**，该实体将从当前用户的联系人记录中检索并在指定的属性字段中设置。  |
| 值             | 在保存窗体时要分派至属性的值。<br>对于“两个选项（布尔）”字段，使用 true 或 false。<br>对于“选项集”字段，使用选项的整数值。<br>对于“查找 (EntityReference)”字段，使用 GUID。<br>**注意**：如果属性也在窗体中，用户的值将被该值覆盖。|
| 从属属性    | 将从当前门户用户的记录检索并在保存时分派给字段的联系人实体中的属性。 |

### <a name="validation"></a>验证

以下部分包含修改各个验证参数和错误消息的属性。

对于为 Common Data Service 环境安装和启用的每种语言包，将提供一个字段用以使用相关语言输入消息。

| 客户                                        | 说明                                                                                                                                                                                                                                                      |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 验证错误消息                    | 覆盖字段的默认验证错误消息。                                                                                                                                                                                                    |
| 正则表达式                          | 将添加以验证字段的正则表达式。                                                                                                                                                                                                          |
| 正则表达式验证错误消息 | 如果正则表达式验证失败将显示的验证错误消息。                                                                                                                                                                               |
| 字段为必填字段                           | 选中使属性字段需要包含值。                                                                                                                                                                                                   |
| 必填字段验证错误消息     | 如果字段不包含值，覆盖默认必填字段错误消息。                                                                                                                                                                        |
| 范围验证错误消息              | 如果字段的值在类型为整数、小数、浮点数或货币的实体属性中指定的相应最大值和最小值之外，将覆盖显示的默认范围验证错误消息。 |
| “地理位置验证器”错误消息         | 适用于：如果属性是单行文本且指定的控件样式为作为地理位置查找验证程序的单行文本，那么当输入验证失败时，这将覆盖显示的默认错误消息。 |
| 固定总和验证错误消息       | 适用于：如果属性是整数类型且指定的控件样式为作为固定总和的组整数，那么当输入验证失败时，这将覆盖显示的默认错误消息。                    |
| 多选验证错误消息    | 适用于：如果属性是两个选项类型且指定的控件样式为多选，那么当输入验证失败时，这将覆盖显示的默认错误消息。                                         |
| 等级次序禁止绑定验证错误消息 | 适用于：如果属性是整数类型且指定的控件样式为作为等级次序禁止绑定的组整数，那么当输入验证失败时，这将覆盖显示的默认错误消息。              |

### <a name="description-and-instructions"></a>说明和介绍

以下属性指定自定义说明或介绍的位置和内容。


|                 客户                 |                                                                                                                                                           说明                                                                                                                                                           |
|--------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|           添加说明            |                                                                                                                        “是”生成显示在指定位置的窗体中的自定义文本。                                                                                                                        |
|               位置               |                                                                                                             以下版本之一：<ul><li>在字段上方</li><li>在字段下方</li><li>在标签上方</li></ul>                                                                                                              |
| 使用属性的说明属性 |                                                                                       选择“是”使用分派给实体中的属性元数据的说明。 选择“否”不提供自定义说明。 默认为“否”。                                                                                       |
|             说明              | 将在窗体中显示的自定义文本。 在“使用属性的说明属性”设置为“否”时结合使用。 对于为 Common Data Service 环境安装和启用的每种语言包，将提供一个字段用以使用相关语言输入消息。 |
