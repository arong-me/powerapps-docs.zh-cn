# <a name="set-personal-options"></a>设置个人选项




更改应用中的显示设置，以满足自己的偏好。 例如，你可以选择希望在登录到应用之后随即显示的页面。 还可以个性化设置多种其他选项，如语言、货币和时区。  
    
## <a name="to-set-personal-options"></a>设置个人选项  
  
1.  选择屏幕右上角的“设置”按钮 ![Power Apps 的选项按钮](media/options-button.png "Power Apps 的“选项”按钮")  。  
  
2.  选择“个性化设置”  。  
  
3.  填写所需信息。   
  
4.  完成后，请选择“确定”  。
