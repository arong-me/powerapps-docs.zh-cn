# <a name="use-keyboard-shortcuts-in-power-apps"></a>使用 Power Apps 中的键盘快捷方式

键盘快捷方式为你提供一种执行常见任务的替代方法，而无需使用鼠标或用手指点击。 借助键盘快捷方式，可在界面无缝导航。 以下键盘快捷方式可应用于应用。 

> [!NOTE]
> 此处介绍的键盘快捷方式指的是美式键盘布局。 其他布键盘局上的键可能与美式键盘上的键不完全一致。 

## <a name="form-shortcuts"></a>窗体快捷方式




|任务  | 统一接口 |
|---------|---------|
|完成有效选项或按钮的命令|  输入|
|取消命令，或关闭选定列表或对话框| Esc|
|保存| Ctrl+S |
|保存并关闭| 不适用|
|取消编辑并关闭（关闭） |Esc|
|删除记录（当窗体处于编辑模式时）| 不适用|
|保存并打开新窗体（保存并新建）（当窗体处于编辑模式时）| 不适用|
|打开查找菜单，按字母顺序显示最近使用的项| 向下键|
|打开查找下拉列表| 输入|
|关闭查找下拉列表|Esc|
|在编辑模式下打开在查找中找到的记录| 输入|
|在电子邮件中添加文章| 不适用|
|应用电子邮件模板（编辑电子邮件时）| 不适用|

## <a name="navigating-submenus"></a>导航子菜单
如果使用屏幕阅读器，请按照以下步骤访问任何子菜单项，将鼠标悬停在导航链接上时会显示子菜单。

1. 使用标准链接导航在菜单选项之间导航。
2. 与平时一样，通过选择 Enter 或空格键来激活链接。 随后将显示子菜单列表。
3. 要访问这些子菜单选项，请使用辅助技术 (AT) 快捷方式导航到下一个按钮。 例如，如果使用 NVDA 屏幕阅读器，可使用“b”快捷键 (NVDA+b)。 要浏览所有选项，请重复选择该键。

> [!IMPORTANT]
> 用户界面 (UI) 的多个区域具有自动折叠的内置菜单。 这包括“导航栏”和“窗体导航”。 选中时两项都会展开，但处于不活动状态 30 秒之后，这两项会折叠。

## <a name="navigating-through-stages-of-a-process"></a>浏览过程的各个阶段
如果你在处理某条记录（例如潜在客户），此部分将显示键盘快捷方式，用于浏览销售过程中为潜在客户授予资格的不同阶段，以及如何在这些阶段中做出选择。 例如，如果希望轻松地从“授予资格”阶段移动到“关闭”阶段，请尝试以下键盘操作：
- 选择一个阶段后，通过选择向下键 (↓) 可转到这些字段。
- 如果阶段为折叠状态，请选择 Enter 将其展开。

  > [!NOTE]
  > 过程名称是位于页面底部而非顶部的图标。

- **Windows 中的键盘快捷方式**：如果使用的是 Windows 操作系统，请选择以下链接，并从下拉列表中选择版本号。 可以查看适用于 Windows 环境的所有快捷方式。 请参阅 [Windows 键盘快捷方式](https://support.microsoft.com/help/12445/windows-keyboard-shortcuts)。 
- **Mac 中的键盘快捷方式**：如果使用的是 Mac 操作系统，请选择以下链接以查看适用于 Mac 环境的所有快捷方式。 请参阅 [Mac 键盘快捷方式](https://support.apple.com/HT201236)。
